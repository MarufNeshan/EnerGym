Gymnasium Management System with fitness tracker Web App

A full-stack web application for tracking fitness progress, BMI, and attendance.

 Features
- User authentication (Admin/User)
- BMI calculator
- Fitness progress tracking
- Attendance system
- Admin dashboard

 Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript

 How to Run
1. Clone the repository
2. Import database file into phpMyAdmin
3. Run using XAMPP server (localhost)
