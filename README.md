Gymnasium Management System with fitness tracker Web App

A full-stack Fitness Tracker & Gym Management web application featuring role-based dashboards for Admin, Trainer, and Users, along with BMI calculation, fitness progress tracking, attendance management, profile management, and gym location management.


Features
- User Authentication System (Admin / Trainer / User)
- Trainer Dashboard
- Admin Dashboard
- BMI Calculator
- Fitness Progress Tracking
- Attendance Management System
- User Profile Management with Photo Upload
- Monthly Attendance Filtering
- Gym Location Management
- Responsive User Interface

  
Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript, Bootstrap

How to Run
1. Clone the repository
2. Import database file into phpMyAdmin
3. Run using XAMPP server (localhost)
